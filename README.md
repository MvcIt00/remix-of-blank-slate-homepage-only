# NXUS Enterprise - Project 2

Gestore flotta, noleggi e contratti per veicoli industriali.

## Tecnologie principali
- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Backend/DB**: Supabase (PostgreSQL)
- **PDF Generation**: @react-pdf/renderer

## Funzionalità principali
- **Gestione Veicoli**: Controllo stato, locazione e storico interventi.
- **Noleggi e Preventivi**: Flusso completo dalla quotazione alla firma del contratto.
- **Generazione Contratti**: Motore PDF professionale con branding aziendale MVC S.r.l.
- **Anagrafiche**: Sistema centralizzato con viste aggregate per sedi e contatti.

## Installazione locale

```sh
# Installa dipendenze
npm install

# Avvia in modalità sviluppo
npm run dev
```
